## **input-remapper 按键映射说明**
### 预设capslock+
大部分capslock+组合键内容抄自capslock+项目关于Linux issue
|`capslock +`           |按下capslock后按下其他键的行为     |
|:-:                    |:-:                                |
|长按capslock           |`do nothing`                       |
|按capslock,150ms内释放 |`capslock`                         |
|                       |                                   |
|`backspace`            |删除整行                           |
|`space`                |`return`                           |
|`return`               |新起一行                           |
|                       |                                   |
|`e/d/s/f`              |`↑/↓/←/→`                          |
|                       |                                   |
|`a/g`                  |向右/左移动一个单词                |
|                       |                                   |
|p                      |`Home`                             |
|;                      |`End`                              |
|                       |                                   |
|w                      |`backspace`                        |
|r                      |`delete`                           |
|                       |                                   |
|q                      |ctrl+w (终端向前删除一个单词)      |
|t                      |alt+d (终端向后删除一个单词)       |
|                       |                                   |
|`z/x/c/v`              |`ctrl+z/x/c/v`                     |
|                       |                                   |
|`i/k/j/l`              |向上/下/左/右选取文字              |
|                       |                                   |
|h                      |左选取一个单词                     |
|.                      |右选取一个单词                     |
|                       |                                   |
|u                      |选取到行首                         |
|o                      |选取到行尾                         |
|                       |                                   |
|                       |下面我也不太确定作用               |
|,                      |开始矩形选区(长按并配合上下左右)   |
|                       |                                   |
|-                      |上翻页                             |
|=                      |下翻页                             |

长按空格不影响与他无关的键输入
|`space +`                      |按下space后的行为                  |
|:-:                            |:-:                                |
|长按space                      |`do nothing`                       |
|按space,200ms内释放            |`space`                            |
|                               |                                   |
|`a/s/d/f/g/h/j/k/l/;/return`   |`1/2/3/4/5/6/7/8/9/0/-/=`          |
|                               |                                   |
|                               |下面其实就是shift+1234……           |
|`tab/q/w/e/r/t/y/u/i/o/p/[/]`  |`~/!/@/#/$/%/^/&/*/(/)/_/+`        |
|                               |                                   |
|n                              |`esc`(vim的normal模式)             |
|                               |                                   |
|v                              |`tab`                              |
|b                              |`shift+tab`                        |
